# 🎧 QuickTunes

QuickTunes is a sleek music discovery app built with React + TailwindCSS, powered by the Shazam Core API via RapidAPI.

## 🚀 Features

- 🔍 Search any song or artist
- 🎨 Dark & Light modes
- 🎧 Preview tracks instantly
- 📱 Fully responsive
- ⚡ Fast with Vite

## 🛠️ Built With

- React (Hooks)
- TailwindCSS
- RapidAPI (Shazam Core)
- Vite
